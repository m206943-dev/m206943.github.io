# Enedis OAuth Site (Static)

This repository contains a **static website** (HTML) you can publish easily with **GitHub Pages**.

## Preview locally
Just open `index.html` in your browser.

## Publish with GitHub Pages (recommended)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Save — GitHub will give you a public URL.

## Notes
- This is a static site (no backend).
- If you later want Vue/Express + OAuth flow, we can convert this into a full stack repo.
